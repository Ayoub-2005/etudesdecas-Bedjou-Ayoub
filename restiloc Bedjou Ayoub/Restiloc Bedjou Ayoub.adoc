== Mission 3 – Application mobile pour les experts

**Question 3.1 :** Écrire en langage SQL les requêtes permettant d'obtenir :

1) Liste des informations date, heure, immatriculation, marque, modèle, expert)

**Réponse :**
[source,sql]
----
SELECT
    M.dateMission,
    M.heureDebut,
    M.immatriculation,
    V.marque,
    V.modele,
    E.nom
FROM MissionExpertise M
JOIN Expert E ON M.idExpert = E.idExpert
JOIN VehiculeExpertise V ON M.idMission = V.idMission
ORDER BY M.dateMission ASC, M.heureDebut ASC;
----
